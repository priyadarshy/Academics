Problem 6.3(c)

num = [1];
den = conv([1 1 0], [0.02 1]);
L = tf(num, den);
% Open Loop Bode
figure;
bode(L);
title('Problem 6.3(c)');
grid on;
% Closed Loop Bode - Phase and Gain Margins
figure;
margin(L/(1+L));
grid on;

Answer

The system is stable becuase of the gain margin being positive.

Problem 6.17(b)

KG = tf(num, den);
figure;
margin(1/(1+KG));
figure;
rlocus(KG);

Answer

We can slide the bode plot up infinitely since we have infinite gain margin, the root locus also shows that positive gain is the only option.
